无需越狱！免费多开/安装第三方应用

如果你是一个有多开需求的用户，那么你大概率已经接触过Testflight或者是从网页直接安装应用了，那么苹果有几种方式安装应用呢，又或者说怎么白嫖呢？
本文将深入浅出的揭秘，或者说带你了解在苹果设备上安装第三方应用的方法，并介绍其利弊。
当然了越详细越啰嗦，你也可以选择观看纯享版
⚠️ 安装第三方应用都会有风险
⚠️ 多开/使用第三方客户端都可能会影响账号使用
⚠️ 不含任何推广以及没有任何一个应用被破解

安装应用的方式

App store

优点

  • 安全可靠，经过苹果审核，保证不会携带已知的病毒或者恶意文件
  • 隐私规范，需要符合苹果的开发者隐私政策

缺点

  • 功能限制多、审核要求高、成本高，部分优质应用或者小工具无法上架
  • 下架后的应用无法安装
  • 需要登陆ID

Testflight（测试平台）

优点

  • 可以体验到部分软件的测试版
  • 上架相对轻松
  • 推送、更新软件相比签名/自签更方便

缺点

  • 安全相对没有保障
  • 安装相对繁琐，需要获取UDID

网页在线安装

通过itms-services协议，将签名好的应用安装到手机

优点

  • 成本相对前两者更低，大范围分发更方便

缺点

  • 签名容易失效，失效后应用无法打开
  • 安全相对更没有保障

自签

分为开发者（收费）/个人（免费），开发者签名有效期1年，个人签名有效期7天
两者都有可能会被ban，但个人账号几率更低

优点

  • 自签的可玩性更高，可以安装自己想要的应用
  • 可以修改应用后再安装，可以实现去广告/破解等等

缺点

  • 操作起来更麻烦，上手需要一定时间
  • 掉签相对会更频繁
  • 个人需要频繁续签

漏洞/越狱

利用漏洞或者越狱后可以自由安装应用，但是众所周知越狱/root就等于放弃了官方给你提供对保障

优点

  • 自由

缺点

  • 对手机型号跟系统版本都有限制
  • 越狱/利用漏洞了就只能自己对自己负责了呢

准备

那么说了这么多废话以后，我们开始进入正题，想要白嫖的话只能选择自签或者是利用漏洞/越狱。

需要准备的

  1. 电脑一台（理论能运行Windows/macOS的设备即可）
  2. Apple ID
  3. iPhone/iPad/iPod Touch（配备Apple Silicon芯片的Mac理论也可以）

工具

  • Sideloadly

    安装更方便/需要依赖电脑

  • AltStore

    安装稍微麻烦一点/需要依赖电脑/在同一个局域网里可以通过Wi-Fi直接续签

  • 爱思助手

    中文界面/安装更方便/需要依赖电脑

  • 牛蛙助手

    中文界面/安装更方便/安装后不需要电脑/安装时需要连接任意Wi-Fi

  • 全能签/轻松签/GBox（签名软件）

    中文界面/安装更方便/不需要电脑/通常配合收费的开发者证书使用

获取软件

  • 使用旧版iTunes/iMazing下载上架AppStore的应用，并使用MacOS砸壳

    比较麻烦，先不写捏

  • 使用dumpapp之类的自动砸壳网站（收费）或者使用已经越狱的设备砸壳（免费）
  • 使用第三方软件源（部分收费）
  • 开发者开发的没有上架的应用

    安全不受保障

方式

  • 白嫖：在第三方软件源下载 → 使用牛蛙助手等自签软件
  • 只需要多开小而美：购买Testflight上架的多开软件
  • 想要更稳定的折腾：购买开发者证书 → 在第三方软件源下载 → 使用签名软件签名
  • 想要稳定不想费心思：购买开发者证书 → 购买收费的第三方软件源 → 使用签名软件签名

上手

为了响应主体，本篇教程以纯白嫖的形式进行演示

安装牛蛙助手

如果你是一个新新新新手可以直接参考视频教学
在电脑浏览器输入牛蛙助手的网址访问官网，并下载对应版本
ios222.com

安装iTunes（Windows）

由于真的挺麻烦，建议下载爱思助手，等显示手机已经连上电脑的时候把爱思助手退出即可

安装到手机

直接运行软件并点击安装直至手机桌面出现牛蛙助手的图标即可

冷知识，macOS用户运行 App 不一定需要放进应用程序文件夹里

配置牛蛙助手

打开牛蛙助手以后需要设置一下牛蛙助手的权限，
牛蛙助手需要的权限有：添加某V配置、使用无线数据、获取UDID（描述文件）

给ipa签名

我们需要把下载好的ipa文件导入进牛蛙助手

牛蛙助手 → IPA签名 → 添加（右上角） → 导入IPA

然后添加自己的Apple ID

牛蛙助手 → IPA签名 → 证书（下方） → 添加（右上角） → 个人证书

添加完成后返回到应用页,选择添加的应用，点击签名
这里选择了之前很火的OldOS，也就是模仿iOS4界面的一个软件
签名设置这里可以修改一些信息，这里主要修改两个地方，第一个是显示名称也就是应用名，第二个是Bundle ID也叫包名，如果把这一栏修改就可以实现应用多开了，但是部分应用修改包名容易出现问题，例如无法实现快速登陆等
修改完成后，我们点击立即签名，签名过程中需要连接Wi-Fi，并保证应用在前台运行
签名完成后我们点击去安装
这里会有我们的一个应用信息以及签名信息
我们再次点击签名后的应用，点击安装，这里提示需要打开某V配置，我们在设置里打开后返回继续安装即可
安装完成后返回桌面，打开应用的时候会提示不受信任的开发者
我们需要在设置里信任一下开发者

设置 → 通用 → 某V与设备管理 → 开发者APP → 信任“你的邮箱”

再次返回桌面就可以运行了
